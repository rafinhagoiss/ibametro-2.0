# Ibametro 2.0

Sistema de inventário e controle de ativos de TI desenvolvido para organizar computadores, responsáveis, setores, status dos equipamentos e chamados técnicos.

## Tecnologias utilizadas

- React Native
- TypeScript
- Firebase
- Firestore
- Expo

## Funcionalidades

- Cadastro de ativos
- Listagem de equipamentos
- Detalhamento de ativo
- Registro de responsável
- Controle de status
- Painel de chamados
- Relatórios básicos

## Objetivo

O projeto foi criado para auxiliar no controle interno de equipamentos de TI, facilitando o acompanhamento de patrimônio, setor, responsável e situação dos ativos.
