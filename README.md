# Ibametro 2.0

Sistema de inventário e controle de ativos de TI desenvolvido para auxiliar na organização, consulta e acompanhamento de equipamentos, responsáveis, setores, status dos ativos e chamados técnicos.

## Sobre o projeto

O Ibametro 2.0 é uma aplicação criada com o objetivo de facilitar o controle interno de patrimônio e equipamentos de tecnologia da informação.  
A proposta do sistema é permitir que a equipe de TI consiga cadastrar, consultar, atualizar e acompanhar ativos de forma mais organizada, reduzindo o uso de controles manuais e centralizando informações importantes em uma única aplicação.

## Tecnologias utilizadas

- React Native
- TypeScript
- Expo
- Firebase
- Firestore

## Funcionalidades

- Cadastro de ativos de TI
- Listagem de equipamentos
- Detalhamento de ativo
- Registro de responsável
- Registro de matrícula
- Controle de setor
- Controle de status do equipamento
- Painel de chamados técnicos
- Relatórios básicos
- Integração com Firebase/Firestore

## Objetivo

O objetivo do projeto é melhorar o controle de equipamentos de TI, permitindo acompanhar dados como patrimônio, tipo de equipamento, setor, responsável, situação do ativo e chamados relacionados.
