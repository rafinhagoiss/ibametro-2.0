# Evidências da visita técnica ao IBAMETRO

Este diretório reúne fotografias da observação em campo realizada para contextualizar o desenvolvimento do projeto. As imagens registram ambientes administrativos, estações de trabalho, infraestrutura de rede e áreas laboratoriais relacionadas ao levantamento das necessidades.

## Cuidados de privacidade

Algumas imagens podem apresentar pessoas identificáveis ou conteúdo exibido em telas. Antes de publicar este repositório na internet, confirme a autorização de uso das fotografias ou disponibilize cópias com rostos e informações sensíveis ocultados.
